# Projet DevSecOps – Secteur Public Marocain

Ce projet présente la mise en œuvre d’un pipeline DevSecOps sécurisé
pour les applications du secteur public marocain.

## Objectifs
- Intégrer la sécurité dès le développement
- Automatiser les scans de vulnérabilités
- Déployer via GitOps sur Kubernetes

## Technologies
- Git / GitHub
- GitLab CI
- Trivy
- Docker & Kubernetes
- ArgoCD
